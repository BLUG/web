Bergen (BSD and) Linux User Group fortsetter møteserien "Last Thursday",

   Torsdag 26. april 2007  kl 19
   Auditorium PI, Autogården (Matematisk institutt), 
   Johannes Bruns gate 12, Bergen[1]

Møtet er gratis og åpent for alle interesserte, med programmet:

The Varnish HTTP accelerator - A tale of high performance programming,
    ved Poul-Henning Kamp

    Varnish (http://varnish-cache.org/) is a high performance HTTP
    accelerator (reverse proxy) sponsored by Verdens Gang and
    published under the BSD license.

    The first part of the talk will introduce Varnish as a
    content delivery tool and introduce some of the features
    that Varnish offers.

    The second part of the talk is about programming for high
    performance and why Varnish is 5-10 times faster than squid.

    Poul-Henning Kamp is a Danish FreeBSD developer, responsible for
    implementation of a vast quantity of systems code, including the
    FreeBSD GEOM storage layer, GBDE cryptographic storage transform,
    part of the UFS2 file system implementation, FreeBSD Jails, and most
    recently, the Varnish HTTP accelerator.

Vi legger som tidligere opp til at den formelle, disiplinerte faglige
delen av møtet vil være fullført ca kl 21.00, med overgang til mer
uformelle aktiviteter og langsom forflytting til egnet serveringssted
